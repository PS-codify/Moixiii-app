# 🌙 Moixiii – Mood Based Playlist Generator

Moixiii is a soft pastel aesthetic web app that generates music playlists based on your mood and vibe.

## ✨ Features
- Select your mood
- Choose your vibe
- Get real song recommendations
- Soft aesthetic UI

## 🛠 Tech Used
- HTML
- CSS
- JavaScript
- iTunes API

## 🚀 How it works
1. Click Get Started
2. Choose your mood
3. Select your vibe
4. Get your personalized playlist

- Add music preview
